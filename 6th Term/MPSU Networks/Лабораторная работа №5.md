## Задание 4: Настройка протокола маршрутизации OSPF

### Router 1:
```
conf term

router ospf 4
router-id 4.1.1.1
network 192.168.4.0 0.0.0.31 area 0
network 192.168.4.64 0.0.0.31 area 0
network 209.165.4.0 0.0.0.3 area 0
passive-interface FastEthernet0/0
exit
  
ip route 0.0.0.0 0.0.0.0 209.165.4.1
router ospf 4
default-information originate
exit

copy run start
```
### Router 2:
```
conf term

router ospf 4
router-id 4.2.2.2
network 192.168.4.32 0.0.0.31 area 0
network 192.168.4.64 0.0.0.31 area 0
network 208.164.4.0 0.0.0.3 area 0
passive-interface FastEthernet0/0
exit

copy run start
```
### Router3:
